# 行业分析引导指南

## 目的

本指南用于指导AI辅助行业分析工作，帮助用户收集目标专业的最新市场数据和行业趋势。

## AI辅助分析流程

### Step 1: 确定分析专业

收集以下信息：
- 专业名称
- 所属专业大类（如：交通运输、信息技术等）
- 区域范围（全国/省市/区域）

### Step 2: 市场数据搜集

使用AI大模型搜集以下信息：

#### 行业概况
- 行业定义与范围
- 行业发展历程
- 行业在国民经济中的地位

#### 行业市场分析
- 行业市场规模
- 市场增长率
- 市场竞争格局
- 市场份额分布

#### 人才需求分析
- 岗位类型分布
- 人才缺口情况
- 薪资水平
- 技能要求趋势

#### 发展趋势
- 技术发展方向
- 政策影响
- 市场需求变化
- 就业前景

### Step 3: 数据分析要点

#### 企业调研维度
| 维度 | 说明 |
|------|------|
| 企业规模 | 大/中/小型企业占比 |
| 岗位设置 | 管理岗/技术岗/操作岗比例 |
| 人才来源 | 学历分布、专业对口率 |
| 技能要求 | 硬技能 vs 软技能 |
| 培训需求 | 入职培训周期、技能提升需求 |
| 发展前景 | 行业生命周期阶段 |

#### 技能需求分析
按照三个层次分析技能要求：

1. **核心技能（5个）**
   - 该岗位最关键的技能
   - 必须具备的能力
   - 直接影响工作绩效

2. **次核心技能（5个）**
   - 重要但非必需的技能
   - 可通过培训获得
   - 影响职业发展

3. **一般技能（5个）**
   - 基础性技能
   - 通用能力要求
   - 职业素养

### Step 4: 生成分析摘要

AI生成的分析摘要应包含：

```
## [专业名称] 行业人才需求分析摘要

### 一、行业概况
（简要描述行业基本情况）

### 二、市场需求
（市场规模、增长率、人才缺口）

### 三、岗位分析
（主要岗位类型、技能要求）

### 四、发展趋势
（技术方向、政策影响）

### 五、建议
（对专业建设的建议）
```

## 调研问卷设计要点

### 企业基本信息
- 企业名称、性质、规模
- 主营业务、产品类型
- 员工总数、专业人员数量

### 人才需求
- 未来1-3年人才需求数量
- 岗位类型分布
- 学历/专业要求
- 薪资范围

### 技能要求
| 技能类别 | 具体内容 |
|----------|----------|
| 专业技能 | 具体的技术能力 |
| 工具使用 | 软件、设备操作 |
| 职业素养 | 责任心、团队合作等 |
| 证书要求 | 职业资格证书 |

### 校企合作
- 合作意愿
- 合作形式建议
- 顶岗实习接收能力

## 数据来源

### 官方数据
- 人力资源和社会保障部数据
- 国家统计局数据
- 上海市教育委员会数据
- 各省市人社厅数据

### 行业数据
- 中国行业协会报告
- 第三方研究机构报告
- 企业年报
- 招聘网站数据（BOSS直聘、智联招聘等）

### AI辅助获取
- 使用AI大模型搜索最新行业报告
- 分析公开的市场研究报告
- 搜集政策文件和发展规划

## 输出格式

生成的分析数据应包含：

1. **数据表格**
   - 企业调研汇总表
   - 岗位技能要求表
   - 人才需求预测表

2. **图表说明**
   - 市场规模趋势图
   - 技能需求分布图
   - 薪资水平对比图

3. **文字分析**
   - 行业现状分析
   - 问题与挑战
   - 发展建议

## 注意事项

1. 数据时效性：优先使用近1-2年的数据
2. 数据来源：标注数据来源，确保可追溯
3. 分析客观性：避免主观臆断，基于数据分析
4. 地域差异：考虑区域经济发展差异
