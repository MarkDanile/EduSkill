# 专业人才需求调研方案编制与报告生成

## Overview

本技能帮助用户完成专业人才需求调研的全流程工作：
1. 通过对话引导用户完成调研方案的制定
2. AI大模型辅助搜集最新市场数据和行业趋势
3. 生成符合学校规范的调研方案和调研报告
4. 支持工作流程审批（专业教研部→教导事务部→教学校长→备案）

## Workflow

### 第一阶段：调研方案编制

#### Step 1: 收集基本信息

通过问答收集以下信息：

| 问题 | 说明 |
|------|------|
| 调研专业名称 | 本次调研针对的专业 |
| 调研背景 | 为什么要开展此次调研 |
| 调研目标 | 希望达到什么目的 |
| 政策依据 | 相关的政策文件依据 |

#### Step 2: 确定调研内容

确认以下主要任务：

1. **行业企业调研**
   - 调研企业数量（建议10家以上）
   - 调研方式：访谈法、问卷法、实地考察、座谈法
   - 调研内容：
     - 行业企业发展趋势
     - 岗位设置情况
     - 典型工作任务
     - 技能要求（5个核心、5个次核心、5个一般）
     - 人才需求情况
     - 企业管理模式
     - 员工来源与培训需求

2. **周边职业学校调研**
   - 调研范围和目的

#### Step 3: 制定实施计划

| 项目 | 内容 |
|------|------|
| 经费预算 | 列出调研工作经费清单 |
| 任务分工 | 工作小组架构、人员组成 |
| 实施步骤 | 学习培训→调研实施→报告撰写 |
| 时间安排 | 各阶段完成时间节点 |
| 预期效益 | 调研工作预期成果 |

#### Step 4: 保障措施

| 保障类型 | 内容 |
|----------|------|
| 组织保障 | 领导小组架构、职责、考核奖惩 |
| 制度保障 | 管理制度、奖惩制度、安全制度 |
| 经费保障 | 经费管理制度 |
| 安全保障 | 安全措施、应急预案 |

### 第二阶段：AI辅助数据收集

在用户确认基础信息后，使用AI大模型辅助：

1. 搜集目标专业的最新市场趋势
2. 分析行业人才需求报告
3. 生成市场分析摘要
4. 提供行业最新发展动态

### 第三阶段：文档生成

按照标准模板生成文档，结构如下：

```
前言（目的、工作目标、政策依据）
一、指导思想
二、调研目的
三、调研内容
   （一）主要任务
   （二）经费预算
   （三）任务分工
   （四）实施步骤
   （五）预期效益
四、保障措施
   （一）组织保障
   （二）制度保障
   （三）经费保障
   （四）安全保障
```

### 第四阶段：审批流程

按照工作流程推送审批：

| 序号 | 任务 | 责任部门 | 责任岗位 |
|------|------|----------|----------|
| 1 | 制定调研实施方案 | 专业教研组 | 专业带头人 |
| 2 | 审核实施方案 | 专业教研部 | 主任 |
| 3 | 审核实施方案 | 教导事务部 | 主任 |
| 4 | 审批实施方案 | 校长室 | 教学校长 |
| 5 | 组织实施调研 | 专业教研组 | 专业带头人 |
| 6 | 分析结果、编制报告 | 专业教研组 | 专业带头人 |
| 7 | 论证调研报告 | 教导事务部 | 主任 |
| 8 | 审核调研报告 | 教导事务部 | 主任 |
| 9 | 备案调研报告 | 教导事务部 | 教学建设管理岗 |

## 工作流程触发

当用户请求以下操作时触发本技能：
- "制定专业人才需求调研方案"
- "编写调研报告"
- "开展人才需求调研"
- "生成调研方案文档"
- "专业设置前的市场调研"

## 输出格式

生成符合学校规范的Word文档，包含：
- 调研方案（含审批签字栏）
- 调研报告
