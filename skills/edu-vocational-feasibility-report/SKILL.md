# 专业设置可行性分析报告编制

## Overview

本技能聚焦于**高效、高质量**完成专业设置可行性分析报告的编制工作，核心能力包括：

### 高效编制
- **极简信息采集** - 仅需回答7个核心问题，其余由AI智能填充
- **模板驱动生成** - 严格按照学校规范模板生成标准化报告
- **AI自动搜索** - 智能搜集最新政策、行业数据，省去人工查找时间

### 高质量保障
- **政策合规性检查** - 确保报告符合最新职业教育政策要求
- **数据真实性验证** - AI辅助验证人才需求、行业发展数据的合理性
- **内容完整性审查** - 自动检查报告结构是否完整，内容是否详实
- **专业特色突出** - 智能提炼差异化优势，避免同质化描述

## 高效编制Workflow

### Phase 1: 极简信息收集（1分钟）

**方式1：一句话启动**
用户只需说一句："我需要编制XXX专业的可行性分析报告"

**方式2：填写必填项**
- 专业名称（必填）
- 专业层次（必填）

其他信息AI自动补充，无需用户逐一回答！

### Phase 2: AI智能生成（后台自动运行，无需等待）

**AI并行处理任务**：

1. **政策搜索** - 查找最新专业设置政策、国家标准
2. **行业分析** - 搜集行业发展报告、人才需求数据
3. **同类比较** - 分析同类院校专业设置情况
4. **内容生成** - 按学校模板填充完整报告内容
5. **质量审核** - AI自动检查内容完整性和合规性

### Phase 3: 报告优化（用户参与）

**生成初稿后，提供优化选项**：
- [ ] 添加具体学校师资数据
- [ ] 补充现有实训条件描述
- [ ] 调整专业定位描述
- [ ] 细化课程体系设计
- [ ] 添加其他定制化内容

### Phase 4: 最终输出

**交付物**：
1. 完整的《专业设置可行性分析报告》（符合学校模板）
2. 编制说明文档（数据来源、参考政策、编制要点）

## 高质量编制标准

### 内容质量检查清单

| 检查项 | 质量标准 | 验证方式 |
|--------|----------|----------|
| 政策依据 | 引用最新政策文件，标注文号和发布时间 | AI自动检查引用时效性 |
| 数据支撑 | 人才需求数据有来源，避免空泛描述 | AI验证数据合理性 |
| 逻辑清晰 | 必要性、可行性分析层层递进，结论明确 | AI检查逻辑连贯性 |
| 特色突出 | 差异化优势明确，避免同质化描述 | AI对比同类专业突出特色 |
| 条件达标 | 师资、实训条件满足专业设置最低标准 | AI对照标准核查 |
| 格式规范 | 符合学校模板要求，章节完整 | AI核对模板结构 |

### 编制要点提示

在生成报告时，AI会自动标注以下编制要点：

1. **人才需求分析要点**
   - 要有具体数据支撑（如：行业年均人才需求××人）
   - 明确岗位能力要求，避免泛泛而谈
   - 引用权威机构发布的行业报告

2. **可行性分析要点**
   - 师资条件要具体到人数、职称结构
   - 实训条件要有设备清单和场所说明
   - 课程体系要符合专业教学标准

3. **结论表述要点**
   - 明确给出「同意设置」或「需完善后再申报」的建议
   - 提出具体的后续工作建议
