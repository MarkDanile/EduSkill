# 专业设置可行性分析报告编制

## Overview

本技能聚焦于**高效、高质量**完成专业设置可行性分析报告的编制工作，核心能力包括：

### 高效编制
- **极简信息采集** - 仅需提供2个必填项，其余由AI智能填充
- **模板驱动生成** - 严格按照学校规范模板生成标准化报告
- **AI自动搜索** - 智能搜集最新政策、行业数据，省去人工查找时间

### 高质量保障
- **政策合规性检查** - 确保报告引用最新、真实的政策文件，标注文号和发布时间
- **数据真实性验证** - 所有数据必须有明确来源，标注发布单位和时间
- **内容详实性要求** - 必要性和可行性论述从多个维度展开，逻辑严密
- **专业特色突出** - 智能提炼差异化优势，避免同质化描述

## 高效编制Workflow

### Phase 1: 极简信息收集

**方式1：一句话启动**
用户只需说一句："我需要编制XXX专业的可行性分析报告"

**方式2：填写必填项**
- 专业名称（必填）
- 专业层次（必填）

其他信息AI自动补充，无需用户逐一回答！

### Phase 2: AI智能生成（后台自动运行，无需等待）

**AI并行处理任务**：

1. **政策搜索** - 查找最新专业设置政策、国家标准（必须真实有效）
2. **行业分析** - 搜集行业发展报告、人才需求数据（必须标注来源）
3. **同类比较** - 分析同类院校专业设置情况
4. **内容生成** - 按学校模板填充完整报告内容（详实、有逻辑）
5. **质量审核** - AI自动检查内容完整性和合规性

### Phase 3: 报告优化（用户参与）

**生成初稿后，提供优化选项**：
- [ ] 添加具体学校师资数据
- [ ] 补充现有实训条件描述
- [ ] 调整专业定位描述
- [ ] 细化课程体系设计
- [ ] 添加其他定制化内容

### Phase 4: 最终输出

**交付物**：
1. 完整的《专业设置可行性分析报告》（符合学校模板，内容详实）
2. 《编制说明文档》（数据来源、参考政策、编制要点）

## 高质量编制标准

### 内容质量要求

| 检查项 | 质量标准 |
|--------|----------|
| **政策依据** | 必须引用最新、真实的政策文件，标注文号、发布单位、发布时间 |
| **数据支撑** | 所有数据必须有明确来源，标注数据发布单位、发布时间、具体数据值 |
| **必要性论述** | 从国家战略、产业发展、区域需求、学校发展四个维度层层递进，逻辑严密 |
| **可行性论述** | 从定位、师资、课程、实训、校企合作五个方面详实论证，每项都要有具体内容 |
| **结论表述** | 基于前面分析给出明确结论，提出具体可行的建议 |
| **专业特色** | 突出差异化优势，避免同质化描述 |
| **格式规范** | 符合学校模板要求，章节完整，逻辑清晰 |

### 编制要点提示

在生成报告时，AI会严格按照以下要点：

1. **人才需求分析要点**
   - 必须引用权威机构发布的行业报告，标注报告名称、发布单位、发布时间
   - 要有具体数据支撑（如：行业年均人才需求××人，标注数据来源）
   - 明确岗位能力要求，避免泛泛而谈，列出具体岗位职责和能力要求

2. **可行性分析要点**
   - 师资条件要具体到人数、职称结构、学历结构、双师型比例
   - 实训条件要有设备清单、场所面积、设备总值等具体数据
   - 课程体系要符合专业教学标准，列出具体课程、学时学分、主要内容

3. **结论表述要点**
   - 基于前面分析，明确给出「同意设置」或「需完善后再申报」的建议
   - 提出具体的后续工作建议，有可操作性
