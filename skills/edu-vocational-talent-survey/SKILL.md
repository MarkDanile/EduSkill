# 专业人才需求调研方案编制与报告生成

## Overview

本技能帮助用户完成专业人才需求调研的全流程工作：
1. **精简对话** - 核心信息采集 + AI智能填充
2. **AI主动搜索** - 自动搜集最新政策、行业趋势、标准规范
3. **智能生成** - 基于AI搜索结果自动填充各章节内容
4. **标准输出** - 生成符合学校规范的调研方案和调研报告

## 对话策略

### 核心信息采集（用户回答）

| 问题 | 说明 |
|------|------|
| 调研专业名称 | 本次调研针对的专业 |
| 调研目的类型 | 专业设置论证/人才培养调整/年度常规调研 |
| 调研目标 | 希望达到什么目的（可选，由AI补充完善） |
| 调研范围 | 区域范围（省/市/全国） |
| 调研企业数量 | 建议10家以上 |
| 调研企业类型 | 企业类型（可多选） |
| 调研时长 | 预计完成时间 |
| 经费预算 | 大致预算范围 |

### AI主动填充（无需用户回答）

AI在收集核心信息后，应主动搜索并填充以下内容：

1. **政策依据** - 最新职业教育相关政策文件
2. **指导思想** - 基于专业特点的指导思想
3. **行业趋势** - 最新行业发展报告、市场数据
4. **调研内容** - 基于专业的具体调研要点
5. **预期效益** - 符合专业特点的预期成果
6. **安全保障** - 针对专业特点的安全措施

## Workflow

### Step 1: 收集核心信息

**必须收集（用户回答）：**
- 调研专业名称
- 调研目的类型
- 调研范围
- 调研企业数量和类型

**可选收集（用户回答或不填）：**
- 调研目标
- 调研时长
- 经费预算
- 人员分工

### Step 2: AI主动搜索

在收集核心信息后，立即执行以下AI搜索：

#### 2.1 政策文件搜索
搜索内容：
- 最新职业教育政策
- 相关专业行业标准
- 地方教育主管部门规定

#### 2.2 行业分析搜索
搜索内容：
- 行业市场规模和发展趋势
- 人才需求状况
- 技能要求变化
- 典型企业案例

#### 2.3 竞争分析搜索
搜索内容：
- 同类院校专业开设情况
- 人才培养模式
- 校企合作案例

### Step 3: 生成文档

#### 前言
**来源：AI生成 + 用户确认**
- 根据调研目的和背景生成前言
- 结合最新政策依据

#### 一、指导思想
**来源：AI自动填充**
- 基于专业特点和行业趋势生成
- 包含职业教育核心理念

#### 二、调研目的
**来源：用户目标 + AI完善**
- 用户提供初步目标
- AI补充完善，形成完整表述

#### 三、调研内容
**来源：AI自动填充**
- 根据行业搜索结果生成具体调研内容
- 包含行业企业调研和职业学校调研

#### 四、保障措施
**来源：AI自动填充**
- 根据专业特点生成安全保障措施
- 包含安全措施、应急预案

### Step 4: 输出确认

生成文档后，请用户确认：
- 内容是否符合需求
- 是否有需要调整的部分
- 审批签字栏信息

## AI搜索指南

### 政策依据搜索

当用户确定专业后，AI应搜索：
```
搜索词："[专业名称]" + "职业教育" + "政策" + 最新年份
示例：新能源汽车检测与维修 职业教育 政策 2024
```

### 行业趋势搜索

搜索词：
```
"[行业名称]" + "市场报告" + "发展趋势" + 最新年份
"[行业名称]" + "人才需求" + "岗位技能" + 最新年份
```

### 专业标准搜索

搜索词：
```
"[专业名称]" + "职业资格证书" + "技能标准"
"[专业名称]" + "专业教学标准" + 最新年份
```

## 文档结构

```
前言（AI生成）
一、指导思想（AI自动填充）
二、调研目的（用户目标 + AI完善）
三、调研内容
   （一）主要任务（AI自动填充）
   （二）经费预算（用户确认）
   （三）任务分工（模板预留）
   （四）实施步骤（AI自动生成）
   （五）预期效益（AI自动填充）
四、保障措施
   （一）组织保障（模板预留）
   （二）制度保障（AI自动填充）
   （三）经费保障（模板预留）
   （四）安全保障（AI自动填充）
```

## 审批流程

| 序号 | 任务 | 责任部门 | 责任岗位 |
|------|------|----------|----------|
| 1 | 制定调研实施方案 | 专业教研组 | 专业带头人 |
| 2 | 审核实施方案 | 专业教研部 | 主任 |
| 3 | 审核实施方案 | 教导事务部 | 主任 |
| 4 | 审批实施方案 | 校长室 | 教学校长 |
| 5 | 组织实施调研 | 专业教研组 | 专业带头人 |
| 6 | 分析结果、编制报告 | 专业教研组 | 专业带头人 |
| 7 | 论证调研报告 | 教导事务部 | 主任 |
| 8 | 审核调研报告 | 教导事务部 | 主任 |
| 9 | 备案调研报告 | 教导事务部 | 教学建设管理岗 |

## 工作流程触发

当用户请求以下操作时触发本技能：
- "制定专业人才需求调研方案"

- "开展人才需求调研"
- "生成调研方案文档"
- "专业设置前的市场调研"

## 输出格式

生成符合学校规范的文档，包含：
- 调研方案（含审批签字栏）
- 调研报告（可选）
