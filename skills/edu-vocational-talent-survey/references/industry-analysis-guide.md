# 行业分析AI搜索指南

## 目的

本指南用于指导AI在收集核心信息后，主动搜索并填充调研方案和报告的相关内容。

## 搜索策略

### 第一步：确定搜索关键词

根据专业名称，自动构建搜索关键词：

```
基础关键词 = 专业名称
扩展关键词 = 相关行业/产业领域
```

### 第二步：分类搜索

#### 1. 政策文件搜索

**搜索目标：** 获取最新职业教育相关政策文件

**搜索词示例：**
```
"[专业名称]" + "职业教育" + "专业设置" + "教学标准" + 2024
"[专业名称]" + "中等职业学校" + "专业教学标准" + 最新年份
"[行业名称]" + "十四五" + "发展规划"
"职业教育" + "产教融合" + "政策" + 最新年份
```

**搜索来源：**
- 中华人民共和国教育部官网
- 上海市教育委员会官网
- 国家职业教育政策文件库
- 各省市人社厅/教育局官网

**提取内容：**
- 最新政策文件名称和文号
- 政策要点和核心要求
- 对专业建设的指导意义

#### 2. 行业分析搜索

**搜索目标：** 获取行业发展趋势和市场数据

**搜索词示例：**
```
"[行业名称]" + "市场报告" + 2024
"[行业名称]" + "产业发展报告" + 最新年份
"[行业名称]" + "人才需求分析" + 最新年份
"[行业名称]" + "技能人才" + "缺口" + 最新年份
"[行业名称]" + "岗位需求" + "薪资" + 最新年份
```

**搜索来源：**
- 中国汽车工业协会
- 各行业协会官方报告
- 智联招聘、BOSS直聘等行业报告
- 艾瑞咨询、易观分析等研究机构
- 企业年报和公开数据

**提取内容：**
- 行业市场规模和增长率
- 行业发展趋势和政策导向
- 人才需求数量和结构
- 岗位类型和技能要求变化

#### 3. 技能标准搜索

**搜索目标：** 获取职业资格和技能标准

**搜索词示例：**
```
"[专业名称]" + "职业资格证书"
"[专业名称]" + "职业技能标准"
"[行业名称]" + "职业分类" + "大典"
"[专业名称]" + "1+X证书" + 最新年份
```

**搜索来源：**
- 人力资源和社会保障部
- 中国就业培训技术指导中心
- 职业技能等级证书评价组织

**提取内容：**
- 相关职业资格证书目录
- 职业技能等级标准
- 1+X证书对应岗位

#### 4. 院校竞争分析搜索

**搜索目标：** 了解同类专业开设情况

**搜索词示例：**
```
"[专业名称]" + "中等职业学校" + "上海"
"[专业名称]" + "高职院校" + "专业建设"
"[专业名称]" + "人才培养方案" + "公开"
"[专业名称]" + "校企合作" + "案例"
```

**搜索来源：**
- 各省市中等职业教育质量年度报告
- 全国职业院校技能大赛官网
- 中国高职高专教育网
- 各学校官网专业介绍

**提取内容：**
- 同类院校专业开设情况
- 人才培养模式特色
- 校企合作典型案例

#### 5. 企业调研参考搜索

**搜索目标：** 了解标杆企业情况

**搜索词示例：**
```
"[行业名称]" + "龙头企业" + "企业名单"
"[行业名称]" + "上市企业" + "年报"
"[行业名称]" + "头部企业" + "人才战略"
```

**搜索来源：**
- 企业官网和年报
- 行业媒体和资讯
- 招聘网站企业信息

**提取内容：**
- 行业头部企业名单
- 企业人才需求特点
- 员工技能要求

#### 通用安全措施
1. 调研前进行安全培训，了解调研注意事项
2. 遵守调研单位的安全管理规定
3. 注意人身和财物安全，保持通讯畅通
4. 保管好调研资料和数据

#### 专业特色安全措施

**工科类专业（如机电、汽车等）：**
1. 进入生产车间需佩戴安全帽
2. 遵守设备操作规程，不随意触碰设备
3. 了解高压电、旋转机械等危险源防护知识

**信息技术类专业：**
1. 注意数据安全和隐私保护
2. 遵守网络安全相关规定

**服务类专业：**
1. 注意人身安全，避免单独调研
2. 保管好调研设备和资料

## 注意事项

1. **时效性**：优先使用近1-2年的数据，确保信息时效性
2. **权威性**：优先使用官方发布的数据和文件
3. **可验证性**：注明数据来源，便于用户核实
4. **客观性**：基于数据进行分析，避免主观臆断
5. **实用性**：结合用户实际需求提供有价值的信息
