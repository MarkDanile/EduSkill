## 一、指导思想

以科学发展观为统领，认真贯彻党的教育方针，全面推进素质教育，全面提高学生素质。以学校人才培养接轨行业企业发展趋势、市场人才需求为指向，实事求是进行行业企业调研，收集行业企业最新信息，推进我校人才培养模式改革、教育模式改革、教育机制创新，着力提高育人效益，彰显学校办学特色。

---

## 二、调研目的

调研的总体目标是收集行业企业各方面的信息，为专业及专业建设方向调整以及人才培养模式改革、课程体系改革、教学内容创新、师资队伍建设、专业内部管理、校企合作模式改革、评价模式改革等方面提供可靠依据。

## 三、调研内容

### （一）主要任务

#### 1. 行业、企业调研

设计好调研提纲和调研问卷，通过以下方式方法开展调研：
- 访谈法
- 问卷法
- 实地考察法
- 座谈法

结合区域产业结构和经济发展趋势对**10家以上**专业相关企业展开调研，调查内容包括：

| 调研内容 | 具体说明 |
|----------|----------|
| 行业企业发展趋势 | 行业发展现状与前景 |
| 企业岗位设置情况 | 主要岗位类型 |
| 岗位的典型工作任务 | 日常工作内容 |
| 岗位技能要求 | 5个核心技能、5个次核心技能、5个一般技能 |
| 人才需求情况 | 用工数量、学历要求 |
| 企业管理模式 | 组织架构、管理制度 |
| 员工来源 | 知识结构、专业水平 |
| 培训需求 | 入职培训、技能提升 |
| 企业文化建设 | 企业价值观、团队氛围 |

#### 2. 周边职业学校调研

通过对周边职业学校进行调研，汲取精华，弃其糟粕。
