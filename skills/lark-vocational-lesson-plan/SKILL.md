# 中高职院校教案写作助手

## 概述

本技能帮助中高职院校教师快速生成符合职业教育要求、严格遵循教学设计规范的个性化教案。基于职业教育教案设计标准和最佳实践，能够根据课程信息自动生成结构完整、内容专业的教学方案。

## 核心工作流程

### 0. 模式选择（第一步）
**询问用户选择教案生成模式：**

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 精准模式 | 精准详细模式，需要用户提供完整材料，生成高质量教案 | 参加比赛、公开课、优质课评比 |
| 粗犷模式 | 快速生成模式，教师提供基本信息即可快速生成 | 日常备课、日常教学 |

### 0.1 教案形式确认（第二步）
**询问用户选择教案呈现形式：**

| 形式 | 说明 | 特点 |
|------|------|------|
| 表格式（默认） | 职业院校最常用的规范格式 | 结构清晰，便于统一备课 |
| 讲义式 | 详细的讲义形式 | 内容详尽，适合理论课 |
| 提纲式 | 简洁的提纲形式 | 重点突出，适合有经验的教师 |

**默认规则**：若用户未明确选择，则自动使用**表格式教案**

### 0.2 基本信息收集（第三步）

**精准模式需要收集的材料：**
1. **人才培养方案**：提取专业培养目标、课程定位、毕业要求等信息
2. **课程标准**：提取课程目标、内容要求、教学安排等信息
3. **电子教材**：提取本节课相关知识点、内容结构

**可选材料（用户可选择上传）：**
1. **授课计划**：提取本课程教学进度安排
2. **行业标准及法规**：提取行业规范要求
3. **证书标准**：提取职业技能等级证书要求

**基本信息格式**：课程名称，课题名称，课时，授课班级，周次，课次

**示例**：护理学基础，生命体征测量，2课时，护理2101班，第6周，第11次课

### 1. 输入处理与信息抽取

**精准模式工作流程**：
- **接收基本信息**：首先询问用户提供基本信息（课程名称、课题、课时、班级、周次、课次）
- **材料收集**：根据用户提供的基本信息，引导上传/输入相关材料
- **信息抽取**：从上传的材料中提取关键信息，与教案内容严格对应

**粗犷模式工作流程**：
- **快速接收输入**：课程名称、课题名称、授课班级、课时信息等基础信息
- **快速生成**：根据基础信息快速生成教案
- **简化流程**：不要求用户提供额外材料，直接基于通用模板生成

#### 材料缺失时的处理方式

当用户未提供完整材料时，大模型基于通用知识进行补充：

| 材料类型 | 处理方式 |
|----------|----------|
| 行业标准 | 根据课程专业类型，自动搜索相关行业规范要求（如护理/机电/计算机等行业标准） |
| 电子教材 | 根据课题内容，大模型根据通用知识补充本节课知识点、内容结构、重点难点 |
| 证书标准 | 根据专业类型，自动关联相关职业技能等级证书要求（如1+X证书、企业认证等） |

- **信息完整度高的材料**：优先使用用户提供的材料内容，确保教案专业性和准确性
- **缺失材料**：大模型基于通用知识补充，保证教案能够完整生成
- **自动课程类型判定**：根据内容确定课程类型（理论课/实训课/理实一体化课）
- **需求分析**：识别用户的具体需求（新授课/复习课/练习课等）

### 2. 资源加载与模板应用
- **加载教案模板**：根据用户选择的教案形式（表格式/讲义式/提纲式）加载对应模板
- **加载设计原则**：应用教案设计原则（系统性、可行性、适应性）
- **加载示例参考**：参考优秀教案示例进行个性化调整

### 3. 教案生成
- **格式确定**：与用户确认教案呈现方式，默认表格式
- **模板填充**：使用标准教案模板作为基础框架
- **职教特色**：融入项目化教学、任务驱动、岗课赛证融合等职教元素
- **思政融入**：在设计意图中体现思政元素和四新融入

### 4. 输出与调整
- **完整教案输出**：生成包含所有必要要素的完整教案
- **教学后记模板**：预留教学纪要和教学反思的填写模板
- **用户反馈调整**：根据用户修改意见进行针对性调整

## 教案结构指南

### ⚠️ 格式规范（必须严格遵守）

**编号规则**：
- 主标题：使用中文数字 一、二、三
- 子标题：使用阿拉伯数字 1. 2. 3.
- 列表项：使用 - 短横线
- 禁止使用：（一）（二）（三）、(1)(2)(3) 等混合格式

**教学分析结构**（学情分析三个维度必须独立分段）：
```
### 1. 内容分析
- 要点1
- 要点2

### 2. 学情分析
#### 知识与技能基础
- 要点

#### 认知与实践能力
- 要点

#### 学习特点
- 要点
```

**教学目标**（统一使用 - 列表格式）：
```
### 3. 教学目标
- 知识目标：xxx
- 能力目标：xxx
- 情感态度价值目标：xxx
```

**教学过程**（只保留统一表格，不需要分散详细说明）：
```
| 教学环节 | 教学内容 | 教师活动 | 学生活动 | 设计意图 |
|---------|---------|---------|---------|----------|
| 导入环节 | | | | |
| 新知讲授 | | | | |
| ... | | | | |
```
**禁止**在表格后面再展开分散的详细说明段落！

## 使用示例

### 场景一：精准模式（高质量教案）

**用户输入**："我需要参加教学能力比赛，请帮我生成一个高质量教案"
**技能响应**：
1. 询问用户选择模式（精准/粗犷）
2. 引导用户提供基本信息（课程名称、课题、课时、班级、周次、课次）
3. 引导上传材料（人才培养方案、课程标准、电子教材、授课计划等）
4. 从材料中抽取关键信息
5. 基于完整材料生成高质量教案

**精准模式示例对话**：
```
技能：请问您需要哪种教案生成模式？
用户：精准模式
技能：请选择教案呈现形式：
      - 表格式（默认）
      - 讲义式
      - 提纲式
用户：表格式（或直接回默认）
技能：请提供基本信息：
      格式：课程名称，课题名称，课时，授课班级，周次，课次
用户：护理学基础，生命体征测量，2课时，护理2101班，第6周，第11次课
技能：请上传材料（可选）
用户：跳过
技能：正在生成表格式教案...
```

### 场景二：粗犷模式（快速生成）

**用户输入**："帮我写一个新能源汽车驱动电机检修的教案，2课时"
**技能响应**：
1. 询问用户选择模式（精准/粗犷）
2. 用户选择粗犷模式
3. 直接基于基础信息快速生成教案
4. 无需上传额外材料

## 注意事项

1. **课程标准遵循**：严格遵循职业教育课程标准和教学大纲
2. **职教特色体现**：突出项目化教学、任务驱动、能力本位
3. **思政元素融入**：在设计意图中体现思政元素和四新融入
4. **个性化调整**：根据学生实际情况调整教学难度和任务设计
5. **信息化融合**：充分利用数字化教学资源和信息化平台
6. **持续改进**：基于教学反思不断优化教案设计
