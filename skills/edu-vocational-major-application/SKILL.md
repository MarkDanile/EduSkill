# 专业设置可行性分析报告编制与审核全流程

## Overview

本技能帮助用户完成专业设置可行性分析报告的编制与审核全流程工作：
1. **精简对话** - 核心信息采集 + AI智能填充
2. **AI主动搜索** - 自动搜集最新政策、行业趋势、同类专业数据
3. **智能生成** - 基于AI搜索结果自动填充10大核心模块
4. **流程指导** - 提供校内校外完整审核流程指引

## 对话策略

### 核心信息采集（用户回答）

| 问题 | 说明 |
|------|------|
| 专业名称 | 拟设置的专业名称 |
| 专业层次 | 中职/高职/中高职贯通/五年一贯制 |
| 学制 | 3年/5年/其他 |
| 招生规模 | 首年计划招生人数 |
| 所属行业领域 | 如：信息技术、智能制造、商贸服务、艺术设计等 |
| 服务区域 | 服务区域（省/市/全国） |
| 专业群归属 | 该专业所属的专业群名称（如有） |

### AI主动填充（无需用户回答）

AI在收集核心信息后，应主动搜索并填充以下内容：

1. **企业社会人才需求** - 行业发展趋势、岗位需求、人才缺口数据
2. **学校发展需求** - 符合学校发展定位的分析
3. **专业群发展需求** - 该专业在专业群中的定位和作用
4. **专业定位** - 培养目标、服务面向、就业岗位
5. **师资条件** - 专业师资、公共基础课程师资、企业特聘教师团队的标准配置
6. **专业建设指导委员会** - 组建方案和运作机制
7. **课程基础** - 课程体系设计、主干课程规划
8. **实训教学条件** - 校内实训基地、教学设备的标准配置
9. **校企合作实训基地** - 典型合作企业和合作模式

## Workflow

### Step 1: 收集核心信息

**必须收集（用户回答）**：
- 专业名称
- 专业层次
- 学制
- 招生规模
- 所属行业领域
- 服务区域
- 专业群归属

**可选收集（用户回答或不填）**：
- 申报类型（目录内专业/目录外专业/国家控制布点专业）

### Step 2: AI主动搜索

在收集核心信息后，立即执行以下AI搜索：

#### 2.1 政策文件搜索
搜索内容：
- 最新职业教育专业设置管理办法
- 2024年新版职业教育专业目录
- 国家/地方教育主管部门规定
- 专业类教学质量国家标准

#### 2.2 行业分析搜索
搜索内容：
- 行业市场规模和发展趋势
- 人才需求状况和缺口数据
- 岗位能力要求和技能标准
- 典型企业案例和用人需求

#### 2.3 同类专业搜索
搜索内容：
- 同类院校专业开设情况
- 人才培养模式比较
- 差异化优势分析

### Step 3: 生成报告

#### 一、专业设置的必要性
**来源：用户信息 + AI搜索生成**
- （一）企业社会人才需求分析
- （二）学校发展需求分析
- （三）专业群发展需求分析

#### 二、专业设置的可行性
**来源：用户信息 + AI自动填充 + AI标准模板**
- （一）专业定位
- （二）师资条件
  - 1．专业师资条件
  - 2．公共基础课程师资
  - 3．企业特聘教师团队
- （三）成立专业建设指导委员会
- （四）课程基础
- （五）实训教学条件
- （六）校企合作实训基地

#### 三、专业设置的结论
**来源：AI综合判断 + 用户确认**
- 可行性综合判断和专业设置结论

### Step 4: 审核流程指导

生成报告后，提供完整审核流程指引：

#### 校内准备与论证阶段
1. 前期调研启动（申报前3-6个月）
2. 可行性报告编制
3. 学院内部审议
4. 校内专家评议
5. 学校决策与公示（≥5个工作日）

#### 校外申报与审批阶段
1. 材料报送（备案/审批）
2. 教育部公示与教指委评议（公示期一个月）
3. 主管部门/教育部审批

#### 后续建设与评估阶段
1. 专业建设实施
2. 动态监测与评估

### Step 5: 输出确认

生成报告和流程指引后，请用户确认：
- 内容是否符合需求
- 是否有需要调整的部分
- 公示和审批时间安排

## AI搜索指南

### 政策依据搜索

当用户确定专业后，AI应搜索：
```
搜索词："专业设置管理办法" + "职业教育" + 最新年份
"[专业名称]" + "职业教育" + "政策" + 最新年份
"职业教育专业目录" + 最新年份
"专业类教学质量国家标准" + "[专业类]"
```

### 行业趋势搜索

搜索词：
```
"[行业名称]" + "市场报告" + "发展趋势" + 最新年份
"[行业名称]" + "人才需求" + "岗位技能" + 最新年份
"[行业名称]" + "人才缺口" + 最新年份
```

### 同类专业搜索

搜索词：
```
"[专业名称]" + "高校" + "专业设置"
"[专业名称]" + "培养方案" + 最新年份
"[专业名称]" + "就业前景"
```

## 关键审核要点

| 审核项 | 核心指标 | 说明 |
|--------|----------|------|
| 需求匹配度 | 与国家战略、区域产业需求契合度 | 人才缺口数据真实性 |
| 办学条件 | 生师比≥18:1、实践教学比例≥50% | 师资、设备、经费达标 |
| 培养质量 | 课程体系合理性、毕业要求达成度 | 参照国家标准 |
| 特色优势 | 差异化发展、服务学校定位 | 避免同质化 |

## 注意事项

1. **预申报制度**（2024年起）：新增专业原则上提前1年预申报
2. **公示要求**：校内公示≥5个工作日，教育部平台公示一个月
3. **建设周期**：通常3-5年，需明确分阶段建设目标
4. **动态监测**：专业建设满3年后接受评估
