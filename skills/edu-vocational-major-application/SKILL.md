# 专业设置可行性分析报告编制与审核全流程

## Overview

本技能帮助用户完成专业设置可行性分析报告的编制与审核全流程工作：
1. **精简对话** - 核心信息采集 + AI智能填充
2. **AI主动搜索** - 自动搜集最新政策、行业趋势、同类专业数据
3. **智能生成** - 基于AI搜索结果自动填充10大核心模块
4. **流程指导** - 提供校内校外完整审核流程指引

## 对话策略

### 核心信息采集（用户回答）

| 问题 | 说明 |
|------|------|
| 专业名称 | 拟设置的专业名称 |
| 专业层次 | 中职/高职/中高职贯通/五年一贯制 |
| 学制 | 3年/5年/其他 |
| 招生规模 | 首年计划招生人数 |
| 所属行业领域 | 如：信息技术、智能制造、商贸服务、艺术设计等 |
| 区域范围 | 服务区域（省/市/全国） |
| 申报类型 | 目录内专业/目录外专业/国家控制布点专业 |

### AI主动填充（无需用户回答）

AI在收集核心信息后，应主动搜索并填充以下内容：

1. **政策依据** - 最新职业教育政策文件、专业设置管理办法
2. **行业趋势** - 最新行业发展报告、市场规模数据
3. **人才需求** - 行业人才缺口数据、岗位能力要求
4. **同类专业** - 国内外相近专业设置情况、差异化分析
5. **培养方案** - 基于国家标准的课程体系设计
6. **办学条件** - 师资队伍、教学设施、图书资料的标准要求
7. **实施计划** - 分阶段建设目标、进度安排
8. **预期效益** - 人才培养质量、社会服务贡献
9. **风险分析** - 政策变化、市场波动、就业风险及应对措施

## Workflow

### Step 1: 收集核心信息

**必须收集（用户回答）：**
- 专业名称
- 专业层次
- 学制
- 招生规模
- 所属行业领域
- 区域范围
- 申报类型

**可选收集（用户回答或不填）：**
- 建设性质（新设/调整/撤销）
- 依托学科
- 预申报年份
- 建设周期

### Step 2: AI主动搜索

在收集核心信息后，立即执行以下AI搜索：

#### 2.1 政策文件搜索
搜索内容：
- 最新职业教育专业设置管理办法
- 2024年新版职业教育专业目录
- 国家/地方教育主管部门规定
- 专业类教学质量国家标准

#### 2.2 行业分析搜索
搜索内容：
- 行业市场规模和发展趋势
- 人才需求状况和缺口数据
- 岗位能力要求和技能标准
- 典型企业案例和用人需求

#### 2.3 同类专业搜索
搜索内容：
- 同类院校专业开设情况
- 人才培养模式比较
- 差异化优势分析

### Step 3: 生成报告

#### 1. 项目总论
**来源：用户信息 + AI生成**
- 专业名称、层次、学制、招生规模
- 建设性质、依托学科

#### 2. 背景与必要性
**来源：AI自动填充 + 用户确认**
- 国家/地方政策导向
- 产业发展趋势
- 人才缺口数据
- 与区域发展的契合度

#### 3. 人才需求分析
**来源：AI搜索生成**
- 行业企业调研数据
- 就业前景预测
- 岗位能力要求

#### 4. 同类专业比较
**来源：AI搜索生成**
- 国内外高校相近专业设置情况
- 差异化优势分析

#### 5. 培养方案设计
**来源：AI自动填充**
- 培养目标
- 知识能力素质结构
- 主干课程
- 实践环节（高职≥50%）

#### 6. 办学条件论证
**来源：AI标准模板 + 用户补充**
- 师资队伍（生师比≥18:1）
- 教学设施（实验室、实训基地）
- 图书资料
- 现有条件评估+缺口分析+建设规划

#### 7. 实施计划与进度
**来源：AI自动生成**
- 分阶段建设目标
- 年度任务分解
- 经费预算

#### 8. 预期效益评估
**来源：AI自动填充**
- 人才培养质量
- 社会服务贡献
- 量化评估指标

#### 9. 风险分析与应对
**来源：AI自动填充**
- 政策变化风险
- 市场波动风险
- 师资流失风险
- 就业风险
- 风险防控措施

#### 10. 结论与建议
**来源：AI综合判断 + 用户确认**
- 可行性综合判断
- 申报建议
- 后续保障措施

### Step 4: 审核流程指导

生成报告后，提供完整审核流程指引：

#### 校内准备与论证阶段
1. 前期调研启动（申报前3-6个月）
2. 可行性报告编制
3. 学院内部审议
4. 校内专家评议
5. 学校决策与公示（≥5个工作日）

#### 校外申报与审批阶段
1. 材料报送（备案/审批）
2. 教育部公示与教指委评议（公示期一个月）
3. 主管部门/教育部审批

#### 后续建设与评估阶段
1. 专业建设实施
2. 动态监测与评估

### Step 5: 输出确认

生成报告和流程指引后，请用户确认：
- 内容是否符合需求
- 是否有需要调整的部分
- 公示和审批时间安排

## AI搜索指南

### 政策依据搜索

当用户确定专业后，AI应搜索：
```
搜索词："专业设置管理办法" + "职业教育" + 最新年份
"[专业名称]" + "职业教育" + "政策" + 最新年份
"职业教育专业目录" + 最新年份
"专业类教学质量国家标准" + "[专业类]"
```

### 行业趋势搜索

搜索词：
```
"[行业名称]" + "市场报告" + "发展趋势" + 最新年份
"[行业名称]" + "人才需求" + "岗位技能" + 最新年份
"[行业名称]" + "人才缺口" + 最新年份
```

### 同类专业搜索

搜索词：
```
"[专业名称]" + "高校" + "专业设置"
"[专业名称]" + "培养方案" + 最新年份
"[专业名称]" + "就业前景"
```

## 关键审核要点

| 审核项 | 核心指标 | 说明 |
|--------|----------|------|
| 需求匹配度 | 与国家战略、区域产业需求契合度 | 人才缺口数据真实性 |
| 办学条件 | 生师比≥18:1、实践教学比例≥50% | 师资、设备、经费达标 |
| 培养质量 | 课程体系合理性、毕业要求达成度 | 参照国家标准 |
| 特色优势 | 差异化发展、服务学校定位 | 避免同质化 |

## 注意事项

1. **预申报制度**（2024年起）：新增专业原则上提前1年预申报
2. **公示要求**：校内公示≥5个工作日，教育部平台公示一个月
3. **建设周期**：通常3-5年，需明确分阶段建设目标
4. **动态监测**：专业建设满3年后接受评估
